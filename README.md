# 🏟️ Turff - Stadium Booking System

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/SakshiNerkar/turf-booking-system)

## 🎯 Features
- **3 Roles**: Admin, Owner, Customer.
- **Booking Flow**: Real-time slot management.
- **Analytics**: Revenue dashboards for owners.
- **Modern UI**: Full-screen maps, animations, and dark mode.

## 🛠️ Tech Stack
- **Frontend**: Next.js, React, Tailwind CSS.
- **Backend**: Node.js, Express, TypeScript.
- **Database**: Supabase / PostgreSQL (Local: SQLite).

## 🚀 Local Setup
1. `cd backend && npm install && npm run dev`
2. `cd frontend && npm install && npm run dev`
